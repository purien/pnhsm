Command	                Comment	                                                    guest
?00	              Version	Y
?01[text]	        Echo text 	                                                Y
?02	                Disconnect	                                                Y
?05[text]	        Set root  psk-identity	                                        N
?06[text]	        Set guest psk-identity	                                        N
?0A	                Get app ID	                                                Y
?0B	                Get Certificate                                         	Y
?0C[64 hexa digits]	Authenticate(32 bytes)	                                        Y
?0E[64 hexa digits]	Set Certificate	                                                N
?A0[64 hexa digits]	Set PSK2 (used with guest)	                                Y
?A1	                Get PSK2 (used with guest)	                                Y
?A2[64 hexa digits]	Set PSK3 used for  TLSSE binder and derive procedure	        N
?D0[64 hexa digits]	Compute the binder procedure with PSK3	                        N
?D1[64 hexa digits]
?D2[64 hexa digits]	Compute the derive procedure with PSK3
                        D2 User must confirm with push button	                        N
?A5[2 hexa digits]	bit 0 (0x01) enable guest psk-identity, 
                        bit 7 (0x80) check server name	                                N
?AA[OldPSK,NewPSK]	Set root PSK OldPSK=64HexaDigit NewPSK=64HexaDigits	        N
?FF[hexa digits]	Echo(hexadecimal value]	                                        Y    
cxy, Cxy	        Clear key at index=xy hexadecimal
                        c for SECP256k1,  C for SECP256r1 (prime 256 curve)	        N
gxy	                Generate SECP256k1, key index=xy hexadecimal	                N
Gxy	                Generate SECP256r1, key index=xy hexadecimal	                N
sxy[64 hexa digits]	Sign value (up to 32 bytes), key index xy hexadecimal	        N
pxy	                Get public key, key index xy hexadecimal	                N
rxy	                Get private key, key index xy hexadecimal	                N
Pxy(130 hexa digits]	Set public key (with prefix 04)
                        65 bytes at key index xy hexadecimal	                        N
Rxy[64 hexa digits]
Xxy[64 hexa digits]	Set private key, 65 bytes at key index xy hexadecimal	        N
txy[hexa digits]
Txy[hexa digits]	Set BIP32 seed (up to 32 bytes) at key index xy hexadecimal
                        Also used by HMAC256, or AES procedures	                        N
kxy[hexa digits]
bxy[hexa digits]	Compute BIP32 key, at key index xy hexadecimal
                        Path is a set of 32 bits value, with MSB set (hardened  key)	N
vxy	                Get BIP32 or seed for HMAC or AES at key index xy hexadecimal	N
*hxy[message]	        Compute HMAC256(seed,message) at index y, x=0,8 (binary),4 pipe Y
*Hxy[message]     	Compute HMAC512(seed,message) at index y,x=0,8 (binary),4 pipe	Y
*Axy[data]	        AES128(seed,data) encryption at index y, x=0,8 (binary),4 pipe	Y
*axy[data]	        AES128(seed,data) decryption at index y, x=0,8 (binary),4 pipe	Y
Zxy[hexa digits]	Write value (up to 32 bytes) in record number xy hexadecimal	N
Ixy	                Read record number xy hexadecimal	                        Y

* h,H,A,a commands have special operation mode (som) see next section
